Unidad 1 — Fundamentos de Estadística Computacional

Resumen rápido

Objetivo: entender conceptos básicos y cómo implementarlos con código.

Temas

  • Definición y rol de la estadística computacional
  • Tipos de datos y medidas resumen: media, mediana, moda, varianza, desviación estándar
  • Visualización básica: histogramas, diagramas de caja (boxplot)
  • Introducción a muestreo y errores

Ejemplo simple (manual + calculadora)

Conjunto de datos: [4, 7, 6, 9, 5]

  • Media = (4+7+6+9+5) / 5 = 31 / 5 = 6.2
  • Mediana = 6 (orden: 4,5,6,7,9)
  • Varianza muestral = suma((x - media)^2) / (n-1) = ... (ejercicio)

Unidad 2 — Simulación y Generadores de Números Aleatorios

Resumen rápido

Temas

  • Generadores congruenciales lineales (LCG)
  • Mersenne Twister (concepto)
  • Pruebas simples de uniformidad
  • Simulación Monte Carlo básica

Ejemplo: LCG simple en JavaScript

Implementación educativa — no usar para criptografía.

// a = 1664525, c = 1013904223, m = 2^32
function lcg(seed){
  var a=1664525, c=1013904223, m=4294967296;
  seed = (a*seed + c) % m;
  return {value: seed / m, seed: seed};
}
Ejercicio 2: Usa el LCG anterior con semilla 12345 y genera 5 números. Anota si parecen uniformes.
